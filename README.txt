# GenreHub: Interactive Movie Genre Showcase Website

GenreHub is an interactive movie genre showcase website that offers users an intuitive and visually appealing interface to browse and explore movies conveniently. Built with HTML for content structure, CSS for styling, and Javascript for dynamic interactivity, GenreHub aims to provide a seamless experience for movie enthusiasts to discover new films across various genres. 

## Features
- Intuitive and visually appealing user interface
- Browse and explore movies conveniently
- Extensive collection of movies across various genres
- Dynamic interactivity for enhanced user experience

## Technologies Used
- HTML
- CSS
- JavaScript
